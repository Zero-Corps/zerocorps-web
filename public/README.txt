Drop your logo here.

1. Save your logo image into THIS folder.
2. Name it exactly: logo.png   (a .png with transparent background looks best)

That's it. The site (index.html) already points to public/logo.png.

If you'd rather use a different name or format (e.g. logo.svg), open
index.html and change this line:

    src="public/logo.png"

to match your file name.
